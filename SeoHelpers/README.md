# SEO Helpers Unlimited

## Use Case 

- Automate technical SEO with [Screaming Frog SEO](https://www.screamingfrog.co.uk/) from the command line. Running in a K8S environment to allow unattended updates. Be able to kick off scanning, save the report, and present.
- Automated building of site maps, link checking, and other command-line checks. One URL or many. Do these with full browser context, as much as possible, to test proxy access.
- Load testing and application regression testing on WordPress staging, including link checking and structure. Be able to map a site without running on production. Ahrefs and Semrush Site Audit is looking at production.
  - Save on licensing fees. Don't duplicate what   [Screaming Frog SEO](https://www.screamingfrog.co.uk/) can do against staging.
  - Bots for regression testing on the website are doing in-house text analysis and data mining. Contextual analysis.

## Scripts

- *bing_url_submission*, Submit the provided URL to the Bing search engine.
- *tool-build-urllist.py*, Traverse a website to create a list of public facing urls.
