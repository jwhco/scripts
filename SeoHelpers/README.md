# SEO Helpers Unlimited

## Use Case 

- Automate technical SEO with Screaming Frog SEO from the command line. Running in K8S environment to allow unattended updates.
- Automated building of site maps, link checking, and other command line checks. One URL or many. Do these with full browser context, as much looking like a human visitor to test proxy access.

## Scripts

- *bing_url_submission*, Submit the provide URL to Bing search engine.
- *tool-build-urllis.py*, Traverse a website to create a list of public facing urls.
