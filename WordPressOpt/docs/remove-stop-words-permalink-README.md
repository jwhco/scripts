# Remove Stop Words for WordPress Permalink

## Overview

- Remove stop words from WordPress post stubs. Works on first creation.
- Stop words are common low value keywords. They are filtered because they have very little SEO value.
- Removing stop words also shortens the overall URL, improve usability for those referencing the content.
- This will NOT change existing permalinks, or slugs, that would need to be done manually. If you change the link, you'll need to redirect the old one.
- Using kebab case, turning `/how-to-write-special-reports/` into `/write-special-reports/` which is shorter.
- Typically removing stop words doesn't significantly change meaning of the phrase.
