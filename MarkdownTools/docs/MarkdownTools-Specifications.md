### Commands

- _extract-names-ner.py_, Use NER to extract PERSON named entities from markdown.
- _topic-modeling-clusters.py_, Report topical clusters in markdown documents.
- _extract-ngram-phrases.py_, Report n-grams (three word default) found in a markdown document.
- _extract-hashtag-terms.py_, Combine both YAML front matter and in body hashtags to list phrases.
